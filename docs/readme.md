# How to use?

### Clone this project
```bash
git clone https://github.com/OmrYvs06/zapret-Turkey-V72.2-Release
```

### copy it to /opt folder
```bash
sudo cp ./zapret-Turkey-V72.2-Release /opt
```

### cd to /opt
```
cd /opt
```

### execute install_easy.sh
```bash
sudo bash ./install_easy.sh
```

next next next to all options (they all configured for turkey) 
and we are done.
